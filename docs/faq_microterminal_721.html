FAQ — Microterminal 721

Descobrir o IP do computador

"Como descubro o IP?", "não sei meu IP", "onde fico o IP do servidor?", "o que é IPv4?", "que número eu coloco?"

▶

Para descobrir o IP do seu computador:

  1. Clique no botão Iniciar do Windows
  2. Digite cmd e abra o aplicativo
  3. Na tela preta, digite ipconfig e pressione Enter
  4. Procure a linha Endereço IPv4 — esse é o número que você vai usar
Anote esse número, ele será necessário na configuração.

Acessar o menu de configuração do terminal

"como entro na configuração?", "não aparece nada quando ligo", "onde aperto o P?", "não aparece os pontinhos", "como reinicio o terminal?"

▶

Para acessar o menu de configuração:

  1. Desligue o microterminal e ligue novamente
  2. Assim que aparecerem os pontinhos na tela, pressione imediatamente a tecla P no teclado
  3. O terminal vai exibir duas opções: IP do servidor e IP do microterminal
atenção O teclado precisa estar conectado antes de ligar o terminal.

"perdi a tela dos pontinhos", "não deu tempo de apertar P", "entrei errado", "como volto para a tela de configuração?"

▶

Sem problema! Basta reiniciar o terminal novamente:

  1. Desligue o microterminal
  2. Ligue e fique com o dedo já posicionado na tecla P
  3. Assim que os pontinhos aparecerem, pressione P imediatamente
Essa janela é bem rápida, então deixe o dedo pronto antes de ligar.

Configurar o IP do servidor

"como coloco o IP?", "onde digito o IP?", "o que faço depois que aparece as duas opções?", "entrei no menu, e agora?"

▶

Depois que o menu aparecer:

  1. Pressione a tecla 1 para configurar o IP do servidor
  2. Digite o IP do seu computador (aquele que você anotou do ipconfig)
  3. Pressione Enter
  4. Pressione a tecla H para acessar o menu de finalização

Salvar e finalizar

"como salvo?", "apareceu salvar e cancelar, o que faço?", "pressionei H e agora?", "não sei como finalizar"

▶

No menu de finalização você verá duas opções: salvar e sair e cancelar.

Pressione a tecla 1 para salvar e sair.

Após isso, aguarde — o terminal vai conectar automaticamente.

Problemas de conexão

"não conectou", "ficou carregando e não fez nada", "salvei mas não conecta", "o terminal não está conectando"

▶

Se após salvar o terminal não conectar, verifique:

  • O IP digitado está correto? (confira novamente com ipconfig)
  • O computador está ligado e conectado à mesma rede?
  • O cabo de rede do terminal está bem encaixado?
Se tudo estiver certo e ainda assim não conectar, entre em contato com o suporte.

"digitei o IP errado", "coloquei o número errado", "preciso corrigir o IP", "como refaço a configuração?"

▶

Para corrigir, basta refazer o processo:

  1. Desligue e ligue o microterminal
  2. Pressione P quando os pontinhos aparecerem
  3. Pressione 1 e redigite o IP correto
  4. Pressione Enter, depois H e depois 1 para salvar
