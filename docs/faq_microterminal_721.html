FAQ — Microterminal 721

Descobrir o IP do computador

"Como descubro o IP?", "não sei meu IP", "onde fico o IP do servidor?", "o que é IPv4?", "que número eu coloco?"

▶

Para descobrir o IP do seu computador:

  1. Clique no botão Iniciar do Windows
  2. Digite cmd e abra o aplicativo
  3. Na tela preta, digite ipconfig e pressione Enter
  4. Procure a linha Endereço IPv4 — esse é o número que você vai usar
Anote esse número, ele será necessário na configuração.

Acessar o menu de configuração do terminal

"como entro na configuração?", "não aparece nada quando ligo", "onde aperto o P?", "não aparece os pontinhos", "como reinicio o terminal?"

▶

Para acessar o menu de configuração:

  1. Desligue o microterminal e ligue novamente
  2. Assim que aparecerem os pontinhos na tela, pressione imediatamente a tecla P no teclado
  3. O terminal vai exibir duas opções: IP do servidor e IP do microterminal
atenção O teclado precisa estar conectado antes de ligar o terminal.

"perdi a tela dos pontinhos", "não deu tempo de apertar P", "entrei errado", "como volto para a tela de configuração?"

▶

Sem problema! Basta reiniciar o terminal novamente:

  1. Desligue o microterminal
  2. Ligue e fique com o dedo já posicionado na tecla P
  3. Assim que os pontinhos aparecerem, pressione P imediatamente
Essa janela é bem rápida, então deixe o dedo pronto antes de ligar.

Configurar o IP do servidor

"como coloco o IP?", "onde digito o IP?", "o que faço depois que aparece as duas opções?", "entrei no menu, e agora?"

▶

Depois que o menu aparecer:

  1. Pressione a tecla 1 para configurar o IP do servidor
  2. Digite o IP do seu computador (aquele que você anotou do ipconfig)
  3. Pressione Enter
  4. Pressione a tecla H para acessar o menu de finalização

Salvar e finalizar

"como salvo?", "apareceu salvar e cancelar, o que faço?", "pressionei H e agora?", "não sei como finalizar"

▶

No menu de finalização você verá duas opções: salvar e sair e cancelar.

Pressione a tecla 1 para salvar e sair.

Após isso, aguarde — o terminal vai conectar automaticamente.

Problemas de conexão

"não conectou", "ficou carregando e não fez nada", "salvei mas não conecta", "o terminal não está conectando", "fica na tela preta", "fica carregando"

▶

Se após salvar o terminal não conectar, verifique:

  • O IP digitado está correto? Rode o ipconfig novamente e confira
  • Atenção: se o computador tem WiFi e cabo de rede, use o IP do cabo (Adaptador Ethernet), não o do WiFi
  • O computador está ligado e no mesmo ramal de rede que o terminal?
  • O cabo de rede do terminal está bem encaixado dos dois lados?
  • Teste no cmd: ping [IP do servidor] — deve aparecer resposta, não "tempo esgotado"
Se tudo estiver certo e ainda assim não conectar, entre em contato com o suporte.

"digitei o IP errado", "coloquei o número errado", "preciso corrigir o IP", "como refaço a configuração?", "errei o IP"

▶

Para corrigir, basta refazer o processo do zero:

  1. Desligue e ligue o microterminal
  2. Pressione P quando os pontinhos aparecerem
  3. Pressione 1 e redigite o IP correto
  4. Pressione Enter, depois H e depois 1 para salvar

"qual IP usar?", "aparece mais de um IP", "tem WiFi e cabo", "endereço IPv4 aparece duas vezes", "qual o IP certo?"

▶

Quando o computador tem WiFi e cabo de rede ao mesmo tempo, o ipconfig vai mostrar mais de um Endereço IPv4.

Use sempre o IP do cabo de rede (aparece sob "Adaptador Ethernet" ou "Ethernet").
Não use o IP do WiFi (aparece sob "Adaptador de Rede sem Fio" ou "Wi-Fi").

O microterminal usa conexão cabeada, então precisa do IP da interface com cabo.

Dificuldades com a tecla P

"não consegui apertar P a tempo", "passou rápido demais", "perdi a janela dos pontinhos", "pressionei P mas não abriu o menu", "não abriu o menu de configuração"

▶

A janela dos pontinhos é muito rápida. Para não perder:

  1. Desligue o microterminal
  2. Antes de ligar, posicione o dedo já na tecla P
  3. Ligue e pressione P imediatamente assim que ligar
atenção O teclado precisa estar conectado antes de ligar o terminal, caso contrário a tecla P não vai funcionar.

Erros e telas comuns

"aparece erro na tela", "aparece um número vermelho", "tela travada", "ficou parado", "não carrega", "aparece Connection refused", "Connection timed out"

▶

Mensagens como Connection refused ou Connection timed out indicam que o terminal não consegue alcançar o servidor.

Causas mais comuns:

  • IP digitado errado — refaça a configuração com o IP correto
  • Software do servidor não está rodando no computador
  • Firewall ou antivírus bloqueando a conexão — verifique se o programa está liberado
  • Computador e terminal em redes diferentes — devem estar no mesmo switch/roteador

"o microterminal desligou sozinho", "apagou a tela", "ficou em stand-by", "tela ficou preta do nada", "parou de funcionar do nada"

▶

O microterminal pode entrar em modo de economia de energia ou perder conexão.

Tente:

  1. Pressione qualquer tecla para "acordar" o terminal
  2. Se não responder, desligue e ligue novamente
  3. Se perder a configuração de IP, faça o processo do P novamente
Se acontecer com frequência, verifique se a tomada ou estabilizador está com problema de fornecimento de energia.

Dúvidas gerais

"o que é o microterminal?", "para que serve?", "como funciona?", "o terminal é um computador?"

▶

O microterminal é um dispositivo que se conecta à rede local (cabo de rede) e acessa o sistema instalado no servidor (computador principal).

Ele funciona como uma extensão do servidor — não processa nada sozinho, apenas exibe e captura dados. Por isso, precisa do IP do servidor para funcionar.

A configuração é feita uma única vez. Depois disso, basta ligar que conecta automaticamente.
